理性决策

Havensli2020/08/14

决策是管理的心脏。

管理学家西蒙指出：“管理就是决策。”

决策是企业管理的核心，它关系到企业的兴衰荣辱、生死存亡。

可以说，领导者科学理性的决策等于成功了一半。

儒佛尔定律：有效预测是英明决策的前提。 吉德林法则：认识到问题就等于解决了一半。 手表定律：别让员工无所适从。 皮尔斯定律：完善培养接班人制度。 羊群效应：提升自己的判断力，不盲目跟风。 自来水哲学：大批量才能生产出廉价产品。 松下水坝经营法则：储存资金，以应付不时之需。 巴菲特定律：到竞争对手少的地方去投资。 吉格勒定理：设定高目标等于达到了目标的一部分。 卡贝定律：放弃有时比争取更有意义。 布利丹效应：成功始于果敢的决策。 普希尔定律：再好的决策也经不起拖延。 沃尔森法则：把信息和情报放在第一位。 哈默定律：天下没有坏买卖。 隧道视野效应：不能缺乏远见和洞察力。 青蛙法则：时刻保持危机意识。 坠机理论：依赖“英雄”不如依赖机制。 奥卡姆剃刀定律：不要把事情人为地复杂化。 帕金森定律：从自己身上找问题。
